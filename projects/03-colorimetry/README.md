# Project 3 - Colorimetry

Implemented spectral-to-XYZ and XYZ-to-xyY calculations and created plots of
standard illuminants, color-matching functions, measured spectra, and chromaticity.

The CIE and ColorChecker reference tables used by `report.m` are stored in
`data/`, while its required MATLAB functions are stored in `code/`. The report
does not depend on the private `course-resources/` directory. Publication of
course-supplied tables or attributed helper functions still requires a separate
redistribution review.
