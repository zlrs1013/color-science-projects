# Project 1 - The Color Problem

Captured a ColorChecker and sample paint patches with an iPhone SE (3rd Gen), 
prepared cropped images in MATLAB, displayed them in a controlled light booth, 
and documented visual differences between real and reproduced colors under the
same viewing conditions.

- `code/`: MATLAB scripts.
- `data/`: original camera capture.
- `results/`: selected crops and visual-comparison photographs.
- `report/`: submitted report.
