# Project 4 - Color Differences

Implemented CIE XYZ-to-CIELAB conversion and Delta E*ab color-difference
calculations, then applied them to ColorChecker and measured-patch data.

The reference datasets used by `project4_report.m` are stored in `data/`; the
report no longer depends on the private `course-resources/` directory.
Publication of course-supplied reference data still requires a separate
redistribution review.
