# Project 3 - Colorimetry

Implemented spectral-to-XYZ and XYZ-to-xyY calculations and created plots of
standard illuminants, CIE 1931 2° Standard Observer color-matching functions, 
measured spectra, and chromaticity.

The CIE and ColorChecker reference tables used by `report.m` are stored in
`data/`, while its required MATLAB functions are located together in `code/`.
