# Fundamentals of Color Science Projects

This repository is a curated presentation of seven projects completed for a course, Fundamentals of Color Science in Fall 2024. The projects progress from observing the color-reproduction problem to measuring color, implementing colorimetric calculations, and characterizing a camera and display for end-to-end color reproduction.

## Project sequence

1. [The Color Problem](projects/01-color-problem/) - Capture and visually
   compare real and imaged ColorChecker colors.
2. [Color Measurement](projects/02-color-measurement/) - Measure spectra,
   CIE XYZ, and CIELAB values for real, imaged, and visually matched colors.
3. [Colorimetry](projects/03-colorimetry/) - Implement spectral-to-XYZ and
   XYZ-to-xyY calculations and visualize CIE data.
4. [Color Differences](projects/04-color-differences/) - Implement CIELAB and
   Delta E*ab calculations.
5. [Camera Characterization](projects/05-camera-characterization/) - Build a
   model that estimates scene XYZ values from camera RGB values.
6. [Display Characterization](projects/06-display-characterization/) - Build
   forward and reverse display models using matrices and lookup tables.
7. [Color Reproduction](projects/07-color-reproduction/) - Connect the camera
   and display models into an end-to-end reproduction workflow.

## Repository organization

Each project is organized into `code`, `data`, `results`, and `report`
directories where those materials are available.
