# Project 1 - The Color Problem

Captured a ColorChecker and sample patches with an iPhone SE, prepared cropped
images in MATLAB, displayed them in a controlled light booth, and documented
visual differences between real and reproduced colors.

- `code/`: readable MATLAB script exported from the original Live Script.
- `data/`: original camera capture.
- `results/`: selected crops and visual-comparison photographs.
- `report/`: submitted report.

The original `.mlx` remains in the private source archive. Run the readable
script directly from its `code/` folder.
